# git的使用

1. 什么是git

5. commit和push的区别
    
    1. 什么是commit
    
        加入到本地仓库
    
    2. 什么是push
     
        将本地仓库的修改内容推送到github    

    3. 什么是pull
    
        从github仓库同步到本地仓库
